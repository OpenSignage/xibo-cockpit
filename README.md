# Xibo Cockpit

Xibo Cockpitは、Mastra Agentと通信するためのWebクライアントアプリケーションです。Agentへの会話インターフェースを提供し、会話履歴の管理やMarkdownによる表示をサポートしています。

## 機能

- Mastra Agentとの会話インターフェース
- ストリーミングレスポンス表示
- Markdownによるテキスト表示
- 会話履歴の管理
- ダークモード対応
- 多言語対応（日本語/英語）

## 使い方

1. パッケージをインストール
```
npm install
```

2. 開発サーバーを起動
```
npm run dev
```

3. アプリをビルド
```
npm run build
```

4. 本番環境で実行
```
npm start
```

## 設定

アプリケーション内の設定ダイアログから以下の項目を変更できます：

- 接続先エンドポイント（デフォルト: http://localhost:4111）
- テーマ（ライト/ダークモード）
- 言語（日本語/英語）
